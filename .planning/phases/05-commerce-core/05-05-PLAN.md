<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| concurrent checkout → inventory | Multiple sessions may race for the last unit of stock. |
| guest cookie → basket | Guest token (random UUID) identifies an unauthenticated basket. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-01 | Tampering | InventoryService.reserveItems | mitigate | `SELECT FOR UPDATE` row lock inside a transaction serializes concurrent reservations; second request reads updated quantity_available (Pitfall 2). |
| T-05-RES | Denial of Service | abandoned reservations | mitigate | ReleaseReservationJob (deterministic jobId, 15-min delay) returns stock; jobId prevents duplicate release jobs on worker restart. |
| T-05-BSK | Spoofing | guest basket token | mitigate | guestToken is a cryptographically random UUID stored in an httpOnly cookie (set in route plan 05-10); BasketService rejects unknown tokens with BasketNotFoundError. |
</threat_model>
