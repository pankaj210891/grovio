<objective>
Create the shared contract layer for Phase 5: basket, checkout, orders, wallet, payments, and commissions type domains. These Zod schemas + inferred types are the single source of truth consumed by both the API (request/response validation) and the storefront (typed React Query data).

Purpose: Establish contracts FIRST so every downstream service and hook builds against fixed shapes (interface-first ordering). Money is always `z.number().int()` minor units at the JSON boundary (Pitfall 5 — bigint never crosses JSON).
Output: Six new contract subfolders, each with `types.ts` + `index.ts`, all wired into the root barrel.
</objective>
