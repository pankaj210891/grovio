## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-V1 | Tampering | CreateProductInputSchema / UpdateProductInputSchema | mitigate | status field omitted from input schemas so vendors cannot set status directly (V5; D-06 enforced downstream via named service methods) |
| T-03-V2 | Tampering | basePriceMinor / priceMinor | mitigate | typed as z.number().int() so float/string price payloads are rejected at parse (V5; BIGINT minor units per D-01) |
| T-03-V3 | Tampering | SuggestQuerySchema.q | mitigate | q is a bounded text value (min 2), never a structured object — prevents query-DSL injection downstream (V5) |
| T-03-V4 | Information Disclosure | SearchHitSchema | accept | search hit fields are public catalog data by design; no PII projected |
</threat_model>
