    Create the data-layer modules. Import React Router hooks from `react-router` (not `react-router-dom`) per Pitfall 8.

    `lib/api-client.ts`: `apiClient` object with `get/post/patch/delete<T>` methods, base URL from `import.meta.env['VITE_API_URL']` (bracket access), every call passing `credentials: 'include'` and `Content-Type: application/json` (D-09). Define and export an `ApiError` class carrying `status` and the parsed error body; throw it on non-OK responses.

    `lib/query-client.ts`: export a `QueryClient` with `defaultOptions.queries` `{ staleTime: 1000*60*2, retry: (count, err) => err instanceof ApiError && [401,403,404].includes(err.status) ? false : count < 2 }`.

    `store/ui-store.ts`: Zustand store holding ONLY `filterDrawerOpen` (+ setter) and `toasts[]` (+ addToast/removeToast). NO filter values here (those live in the URL — D-06 boundary, Pitfall 1).

    `hooks/useFilterState.ts`: `useSearchParams`-based hook exposing `{ filters: { q, categoryId, sort, activeFilters }, setFilter, clearFilters }`; `activeFilters` is the JSON-parsed single `filters` param (avoid URL explosion); `setFilter` uses `{ replace: true }`.

    `hooks/useProductSearch.ts`: `useInfiniteQuery` reading q/filters/sort/categoryId from `useSearchParams`; queryKey `['search', q, filters, sort, categoryId]` (all params in key so filter changes reset to page 1 — Pitfall 1); queryFn calls `GET /search?...&limit=24&cursor=...` (24 per D-08); `getNextPageParam` returns `lastPage.nextCursor ?? undefined`; `initialPageParam: ''`.

    `hooks/useInfiniteScroll.ts`: wraps `useInView` from `react-intersection-observer` (threshold 0.1); calls `fetchNextPage()` in a useEffect when `inView && hasNextPage && !isFetchingNextPage`; returns `{ sentinelRef }`.

    `hooks/useAuth.ts`: exposes auth state via a `useQuery(['session'])` calling `GET /account/profile` (or a `/auth/me` style probe — use `/account/profile`, treating 401 as unauthenticated) plus `login`, `logout`, `signup` mutations calling the api-client; httpOnly cookies are NOT read in JS (Pitfall: detect auth via the session query / 401, never by reading cookies). Expose `{ isAuthenticated, customer, login, logout, signup, isLoading }`.
